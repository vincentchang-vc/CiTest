Comparing Revisions in the Timeline View

To compare two revisions in the timeline view:

  1. Select the first revision by clicking the revision in the timeline.
  2. Select the second revision by holding down the ⌘ key while clicking the revision.
  3. The files will be exported and compared in the compare view underneath the timeline. The earlier revision is always displayed on the left.

Directory comparison is not supported in the current version of Cornerstone.

Comparing using an External Tool

To compare two revisions using an external compare tool:

  1. Select the first revision by clicking the revision in the timeline.
  2. Select the second revision by holding down the ⌘ key while clicking the revision.
  3. Select File > Compare Using… > Compare Selected Revisions (key equivalent ⇧⌘C).
