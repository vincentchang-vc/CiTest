Exporting Revisions from the Log View

To export a revision of a file listed in a revision's change set:

  1. Select the file in the revision's change set.

  2. Select File > Export… or click the Export button in the toolbar:

  3. Select the folder to export the revision(s) to.
  4. Click Export to export the revisions to the selected folder.

You can export both files and folders using the log view.
