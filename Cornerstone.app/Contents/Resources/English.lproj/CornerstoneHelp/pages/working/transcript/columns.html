Transcript View Columns

Transcript entries are displayed in 6 columns:

Time

The local system time when the entry was recorded.

Status

The status of the operation. This can be one of the following statuses:

The operation is running
The operation completed successfully
The operation failed
The entry is a user note

Item Action

The action performed on the file or folder during update and export operations.

This can be one of the following actions:

The item was added
Merging changes resulted in a conflict
The item was deleted
The item already existed
Changes were merged without conflicts
The item was replaced
The item was updated to a newer revision

Property Action

The action performed on the item's properties. The possible actions are as for item actions.

Lock Action

The action performed on the item's lock during update and export operations.

This can be one of the following actions:

The lock was broken or stolen

Description

Textual description of the operation performed.
